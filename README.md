# 장비 / 작업실 / CDP / 포터블 시스템 기준

이 프로젝트에서 장비 평가, 점수 산정, 재배치, 문서 정리는 아래 인수인계서를 최우선 기준으로 삼는다.

- 기준 문서: `장비_작업실_CDP_포터블_시스템_인수인계서.md`
- 최신 작업실 목록 원문: `작업실_목록_최신_원문.md`
- 과거 대화나 이전 해석과 충돌하면 기준 문서를 우선한다.
- 장비 실물 목록을 확인할 때는 최신 작업실 목록 원문을 함께 본다.
- 원문 목록과 기준 문서의 해석이 다르면, 원문은 목록 확인용으로 보되 평가는 기준 문서의 정정 사항을 우선한다.
- `[ ]` 표기는 미보유, 배송중, 구매예정으로 단정하지 않는다.
- CDP 목록의 `→`는 직렬 연결이 아니라 주력 / 보조 이어폰 매칭으로 해석한다.
- 장비 평가는 공식 매뉴얼, 제조사 자료, 영어권 / 한국어권 리뷰, 사용자 장기 후기를 최대한 확인한 뒤 판단한다.
- 근거가 부족한 부분은 추측으로 단정하지 않고 보류 또는 확신도 낮음으로 표시한다.
- 점수는 룸 실측 점수가 아니라 공간 목적, 장비 역할, 신호 흐름, 운용 잠재력, 컬렉션 서사를 함께 보는 복합 평가다.

현재 최종 판정은 장비 추가보다 역할 고정, 라우팅 문서화, CDP 매칭 확정, 운용 프리셋 정리가 더 중요하다는 것이다.
