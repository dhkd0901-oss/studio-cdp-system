목적: ChatGPT에서 진행해온 장비 구성, 작업실 역할, CDP 매칭, 점수 산정, 재배치 기준을 Codex 또는 다른 AI 작업 환경에서 이어받기 위한 기준 문서

# 0. 프로젝트 성격

이 프로젝트는 단순 장비 목록 정리가 아니다.

사용자는 음악 제작, 녹음, 믹스, 마스터링, 하이파이 감상, 포터블 CDP 수집, 오픈형 이어버드 매칭을 하나의 큰 시스템으로 관리한다.
따라서 장비를 평가할 때 단순히 가격, 브랜드, 체급만 보면 안 된다.

반드시 다음 기준을 함께 봐야 한다.

1. 해당 공간의 목적
2. 장비의 실제 역할
3. 신호 흐름의 자연스러움
4. 중복 장비가 불필요한 중복인지, 역할 분리인지
5. 녹음 / 제작 / 믹스 / 마스터 / 감상 / 외부 작업실 / 포터블 감상의 흐름
6. CDP와 이어폰의 출력, 임피던스, 감도, 시대감, 감성 서사
7. 사용자가 중요하게 여기는 “완성도”, “컬렉션성”, “서사”
8. 공식 매뉴얼, 제조사 자료, 영어권·한국어권 리뷰, 사용자 장기 후기 기반의 근거

사용자는 “대충 비슷해 보이니까 묶는 것”을 싫어한다.
모델명, 세대, 단자, 출력, 튜닝, 실제 리뷰, 공식 사양을 최대한 확인해야 한다.

# 1. 응답 원칙

## 1-1. 말투와 평가 방식

사용자는 돌려 말하지 않는 솔직한 평가를 원한다.

좋으면 좋다고 말하고, 애매하면 애매하다고 말한다.
다만 근거 없이 과감하게 단정하면 안 된다.

예시:

* “이건 좋다. 이유는 역할이 겹치지 않고 신호 흐름이 자연스럽기 때문이다.”
* “이건 애매하다. 공식 자료가 부족해서 음색 점수는 보류하는 게 맞다.”
* “이 조합은 감성적으로는 좋은데, 출력 매칭은 최상은 아니다.”
* “이 장비는 음질 업그레이드 목적보다 컬렉션 가치가 크다.”

## 1-2. 절대 하면 안 되는 것

* `[ ]`를 무조건 미보유 또는 구매예정으로 단정하지 말 것.
* 사용자가 직접 존재를 확인한 제품을 “없는 제품” 취급하지 말 것.
* 비슷한 모델명을 임의로 같은 제품으로 합치지 말 것.
* CDP 이어폰 매칭의 화살표를 직렬 연결로 해석하지 말 것.
* 빈티지 CDP의 음색을 근거 없이 너무 정확한 숫자로 단정하지 말 것.
* 출력이 강하다고 저음이 많다고 단정하지 말 것.
* 공식 주파수응답 `20Hz–20kHz`만 보고 음색을 판단하지 말 것.
* 오픈형 이어버드의 저역 평가를 절대값처럼 단정하지 말 것. 착용과 폼 영향이 크다.
* 리뷰 한두 개만 보고 “전부 그렇다”고 말하지 말 것.
* “비싼 장비가 많으니 무조건 100점”처럼 평가하지 말 것.

## 1-3. 점수 산정 기준

점수는 실측 룸 어쿠스틱 점수가 아니다.

점수는 다음의 복합 평가다.

* 장비 구성 완성도
* 공간 목적 적합성
* 최적 연결을 했을 때의 운용 잠재력
* 신호 흐름의 논리성
* 중복 장비의 역할 분리
* 실사용 동선
* 컬렉션 서사
* 향후 확장성

즉, 99점대 점수는 “룸 측정이 완벽하다”는 뜻이 아니라,
“현재 장비 구성과 역할 배치가 목적에 매우 잘 맞는다”는 뜻이다.

# 2. 표기 해석

## 2-1. `[ ]` 표기

`- [ ]`는 단순 체크박스 또는 상태 표기일 수 있다.
무조건 미보유, 배송중, 구매예정으로 단정하지 않는다.

## 2-2. `■` 표기

`■`는 보유 표시로 보는 것이 자연스럽다.
다만 사용자가 따로 정정하면 그 정정을 우선한다.

## 2-3. 화살표 `→`

CDP 목록에서 화살표가 여러 개 있을 경우:

예:

FiiO DM15 R2R
→ Ucotech ES-P3 V2
→ DUNU Alpha3

이것은 직렬 연결이 아니다.
해당 CDP의 주력 / 보조 이어폰 매칭이다.

# 11. 주요 정정 및 주의사항

## 11-1. Maselec MDS-2

MDS-2는 멀티밴드 컴프레서가 아니라 HF limiter / de-esser로 보는 것이 정확하다.

## 11-2. EBX25Ti LUDENS

EBX25Ti LUDENS는 일반 EBX25Ti의 한정 디자인판에 가깝다.
현재 공식 자료상 드라이버나 튜닝 변경 근거는 없다.

따라서:

* 음질 평가는 일반 EBX25Ti 리뷰를 참고 가능
* 단, “LUDENS 전용 리튜닝”이라고 말하면 안 됨
* 차이는 외관, 코지마 프로덕션 협업, 수집 가치, 패키징 중심

## 11-3. D-E01 출력

D-E01은 일반판 기준 출력이 강한 편이다.
프랑스 제한판은 출력이 극히 낮을 수 있으므로 지역판 확인 필요.

## 11-4. D-EJ2000 / D-EJ1000 출력

둘 다 일반판은 약 5mW+5mW/16Ω 수준으로 보고,
일부 유럽/프랑스 제한판은 훨씬 낮은 출력일 수 있다.

따라서 PAVANE, ES-P3 V2 같은 출력 요구가 있는 이어버드는 피하는 편이 안전하다.

## 11-5. 오픈형 이어버드 평가

오픈형 이어버드는 착용, 폼, 귀 모양, 이어솜 두께에 따라 저역이 크게 바뀐다.
저역 점수는 항상 유동적으로 봐야 한다.

## 11-6. 빈티지 CDP 음색 점수

빈티지 CDP는 개체 상태에 따라 음질 차이가 크다.

영향 요소:

* 픽업 상태
* 콘덴서 노화
* 배터리 접점
* 라인아웃 / 헤드폰아웃 차이
* 리모컨 사용 여부
* AVLS 제한
* 지역판 출력 제한
* G-Protection / Anti-shock 모드

따라서 세부 대역 점수는 항상 ±0.3~0.5 정도 오차를 열어둔다.

# 13. Codex가 답변할 때의 기본 방식

장비 평가를 요청받으면 다음 순서로 답한다.

1. 최신 목록 기준으로 해당 장비 위치 확인
2. 방의 목적 확인
3. 공식 스펙 / 매뉴얼 / 리뷰 자료가 있는지 확인
4. 역할 중복인지 역할 분리인지 판단
5. 점수 산정
6. 수정안이 있으면 최소 변경안부터 제시
7. 사용자가 원하면 전체 재배치안 제시

CDP 매칭을 요청받으면 다음 순서로 판단한다.

1. CDP 출력 확인
2. 이어폰 임피던스 / 감도 / 구동 난도 확인
3. 이어폰 성향 확인
4. CDP 성향은 리뷰 공통점만 반영
5. 빈티지 CDP 상태 변수 언급
6. 주력 / 보조 매칭 구분
7. 컬렉션 서사 고려
8. 최종 조합 제시

점수만 요청받으면 표로 간결하게 제시한다.
분석을 요청받으면 근거를 붙여 길게 설명한다.
“미친듯이 분석”이라고 하면 공식 자료, 영문 리뷰, 한국어 후기, 사용자 장기평까지 가능한 범위에서 모두 훑고, 확신도까지 표시한다.
